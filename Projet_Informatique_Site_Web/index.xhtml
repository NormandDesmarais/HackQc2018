Installation du logiciel

Introduction

Le logiciel gratuit et open source Logisim permet d’effectuer la simulation de circuits électroniques allant de circuits très simples à très complexes. C’est un outil puissant pour la conception, mais aussi très intéressant pour l’apprentissage des concepts électroniques. Le logiciel permet d’observer le comportement de différentes composantes électroniques et d’implémenter des circuits utilisés au cœur des processeurs de nos ordinateurs.

Conditions

Pour installer le logiciel, vous devez avoir Java d’installer sur votre machine.

Lien : Java SE

Ensuite, vous devez télécharger Logisim lui-même :

Lien : Logisim

Malheureusement, le logiciel n’est plus mis à jour et date un peu, surtout si vous êtes habitué de travailler avec des logiciels modernes. Toutefois, cela n’enlève rien aux multiples fonctions disponibles.

Fenêtre de base : Projet vide

Le logiciel s’ouvre avec une fenêtre vide. Remarquez à gauche en haut le circuit main est sélectionné. Il est possible de créer plusieurs circuits à l’intérieur d’un même projet et de les renommer afin de les classer logiquement. Nous verrons plus tard qu’il est possible de réutiliser ces « sous-circuits » dans notre circuit principal.

Il y a donc un projet contenant un ou plusieurs circuits (dont le circuit principal). Le circuit sélectionné est affiché dans la fenêtre. Nous allons donc utiliser ces trois termes de façon précise: projet, circuit et fenêtre.

Double-cliquer pour changer le circuit à afficher dans la fenêtre.

Portons maintenant notre attention sur la section du logiciel en bas à gauche. Il s’agit de l’endroit où se trouvent les différents attributs de ce qui est sélectionné. Il est possible, entre autres, de renommer les circuits à cet endroit.

Différentes composantes ont été rajoutées à notre circuit de base! Regardons dans le prochain tutoriel (« Outils de base ») ce qu'il se passe.
