Acclimate

L'application

Description

Acclimate vise à améliorer la réception des alertes lancées par certaines bases de données (actuellement du Ministère de la Sécurité Publique) à l’aide d’une WebApp et d’une application Android. Permet aux usagers de visionner le positionnement de ces alertes ainsi que d’en déclarer. Les alertes sont catégorisées pour aider le filtrage.
